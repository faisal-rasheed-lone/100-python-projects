# Contact Book

The Contact Book is a Python-based digital contact management application that allows users to create, organize, edit, and manage their contacts. This application uses object-oriented programming (OOP) principles to implement contacts, groups, and contact management functionalities.

## Features

- Add new contacts with names, phone numbers, and email addresses.
- Organize contacts into custom groups.
- Remove contacts from the contact book.
- Create and manage contact groups.
- Add contacts to specific groups.
- Remove contacts from groups.
- Display the list of contacts.
- Display the list of contact groups.
- User-friendly command-line interface.

## Usage

1. Clone or download this repository to your local machine.

2. Open a terminal or command prompt and navigate to the project directory.

3. Run the following command to start the Contact Book application:

   ```bash
   python contact_book.py
   ```

4. Follow the on-screen menu to interact with the Contact Book.

## Menu Options

- **Add Contact (Option 1)**: Enter details for a new contact, including name, phone number, and email.

- **Remove Contact (Option 2)**: Remove a contact from the contact book by specifying their name.

- **Create Group (Option 3)**: Create a new contact group with a custom name.

- **Add Contact to Group (Option 4)**: Add a contact to a specific group by providing their name and the group name.

- **Remove Contact from Group (Option 5)**: Remove a contact from a group by providing their name and the group name.

- **Display Contacts (Option 6)**: Display the list of all contacts in the contact book.

- **Display Groups (Option 7)**: Display the list of all contact groups.

- **Exit (Option 8)**: Exit the Contact Book application.

## Getting Started

To get started with the Contact Book application, simply follow the on-screen menu options. You can create contacts, groups, and organize your contacts efficiently.
